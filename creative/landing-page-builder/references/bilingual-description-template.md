## Writing Tips

- Lead with specific numbers: "50+ providers" not "many providers"
- Use technical terms: "IntersectionObserver API" not "scroll animation"
- Mention real tools/libraries by name
- Keep both versions structurally identical
- The "Impact" section should mention performance (load time, size, dependencies)
- Always end with the tool usage + token statement (user's standard closing)

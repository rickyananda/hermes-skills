# MIMO Project Submission Format

When user says "buat deskripsi project buat MIMO" or "submit ke MIMO", use this format.

## Structure (English Version)

```
📝 DESKRIPSI PROJECT - [PROJECT NAME]

---

English Version:

I have built a [one-line description] demonstrating [skills showcased].

Project: [Project Name] — [Subtitle]

[2-3 sentence description of what the project does and the core problem it solves.]

**Core Features & Logic Flow:**
1. [Feature 1 with brief explanation]
2. [Feature 2 with brief explanation]
3. [Feature 3 with brief explanation]
... (6-10 features)

**Technical Implementation:** Built with [Tech Stack]. Features include:
- [Technical feature 1]
- [Technical feature 2]
- [Technical feature 3]
... (4-6 technical features)

**Impact:** [1-2 sentences on value, use cases, and who benefits]

**GitHub:** https://github.com/[username]/[repo]
**Live Demo:** https://[username].github.io/[repo]/

I use Hermes Agent, Claude Code, and Cursor daily for [relevant activities]. I primarily rely on MiMo, Claude, and DeepSeek models for generating clean, production-ready code.
```

## Structure (Chinese Version - 中文版)

```
Chinese Version (中文版):

[Chinese translation of the one-line description]

**项目：[Project Name] — [Subtitle]**

[Chinese translation of the 2-3 sentence description]

**核心功能与逻辑流：**
1. [Chinese translation of feature 1]
2. [Chinese translation of feature 2]
... (same features as English)

**技术实现：** [Chinese translation of tech stack]. 功能包括：
- [Chinese translation of technical feature 1]
- [Chinese translation of technical feature 2]
... (same features as English)

**影响：** [Chinese translation of impact]

**GitHub:** https://github.com/[username]/[repo]
**在线演示:** https://[username].github.io/[repo]/

我每天使用 Hermes Agent、Claude Code 和 Cursor 进行[relevant activities]。我主要依赖 MiMo、Claude 和 DeepSeek 模型生成干净、生产就绪的代码。
```

## Key Points
- Always provide BOTH English and Chinese versions
- Highlight specific numbers (30+ products, 6 features, etc.)
- Mention technical depth (CSS techniques, APIs, frameworks)
- End with tool usage statement (Hermes Agent, Claude Code, Cursor + MiMo, Claude, DeepSeek)
- Keep features as numbered list, technical features as bullet list
- Impact should be 1-2 sentences on real-world value

# Xiaomi MiMo 100T Token Program

## Overview
Xiaomi is giving away 100 trillion tokens for their MiMo AI platform. Free tokens for developers.

## Details
- **URL**: https://100t.xiaomimimo.com/
- **Duration**: 30 days (April 28 - May 28, 2026)
- **Target**: Global users, individual developers, teams, enterprises
- **Platform**: Xiaomi MiMo API (https://platform.xiaomimimo.com)

## How to Apply
1. Visit https://100t.xiaomimimo.com/
2. Click "立即申请" (Apply Now)
3. Fill in the form with:
   - Email address
   - AI tools you use
   - Project description
   - Supporting materials
4. Wait for evaluation (up to 3 days)
5. Check email for approval notification

## What You Get
- Token Plan (subscription-based for Claude Code, Cursor, etc.)
- Or account credits (for pay-per-use API calls)
- Specific tier depends on evaluation

## Important Notes
- Email must match your Xiaomi account registration
- If registered with phone number, bind email first
- Different tiers based on project quality and AI tool usage
- Tokens have expiration dates
- Can be used with Claude Code, Cursor, OpenClaw, etc.

## API Platform
- **Base URL**: https://token-plan-sgp.xiaomimimo.com/v1
- **Model**: mimo-v2.5-pro
- **Compatible with**: OpenAI API format
